# Asset Attribution

Pixel art sourced from the **Tuxemon** project (https://github.com/Tuxemon/Tuxemon),
licensed CC BY-SA 3.0 / GPL-3.0. Used for a non-commercial personal portfolio.

- `prototyping_outdoor.png` — Tuxemon `mods/tuxemon/gfx/tilesets/prototyping_outdoor.png`
  (grass/path autotiles, pines, round trees, fence, signs, foliage, shrub)
- `core_buildings.png` — Tuxemon `mods/tuxemon/gfx/tilesets/core_buildings.png`
  (red-roof cabin and the blue/yellow/red/green storefronts)

The player/NPC overworld trainers, the stone fountain and the welcome sign board
are original pixel art generated programmatically in `scripts/pack-assets.py`
(Tuxemon has no overworld walk sheets).

`scripts/pack-assets.py` slices and composites these into `public/assets/`.
